    Introduction

    Tranquility CRM is a CRM specifically designed for use in outbound call centers.

    Credits

    Tranquility CRM utilizes the following open-source libraries:

    • Laravel
    • Bacon QR Code Generator
    • libphonenumber for php
    • Guzzle for PHP
    • Google 2FA for Laravel
    • Predis
    • AWS SDK for PHP
    • Google API PHP Client
    • Twilio SDK for PHP
    • Laravel Debug Bar
